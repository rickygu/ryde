== README 

Code Challenge Dec 14 2015

Design a data model for Ryde (an Uber competitor) fulfilling the following business needs:

* Enable Ryde to track drivers and cars and car types
* Enable Ryde to track when drivers are available for picking up passengers
* Enable riders to book a ride
* Enable multiple riders to book a shared ride for car types that allow +1 capacity


To run:

    bundle install
    rake db:migrate RAILS_ENV=test
    guard
    
Time spent: 2.5 hours
